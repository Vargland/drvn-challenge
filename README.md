# DrvnChallenge

This project is a challenge for drvn - moveo.

## Stack: 
    - Angular 19.
    - RxJs.
    - SaSS.
    - node 22.12.0

## To run: 
    - git clone.
    - npm install.
    - npm start.
    - Go to: http://localhost:4200/
    - Enjoy

## Running unit tests
    - npm run test.
